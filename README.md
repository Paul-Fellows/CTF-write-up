# Get to Know Me

Hey, I'm Paul 
I've been working in IT for a while now, currently as a Field Site Administrator supporting
DOJ infrastructure across multiple sites in New York and Philadelphia. I'm the only IT person on
site, which means I handle everything from troubleshooting endpoints to keeping the network
running. It's given me a solid foundation but I've always been drawn to the security side of
things, understanding how systems get compromised, how attackers think, and how to defend
against it.
That curiosity is what pushed me toward cybersecurity. I started stacking certs, enrolled in
a B.S. program at WGU for Cloud and Network Engineering, and started doing hands-on labs to
build real skills on top of the theory. The more I got into it the more I realized this is
where I want to be long term. Not just IT security.
I'm using this repo to document that journey. Every writeup here is a real solve from a real
lab. I write about what I tried, what didn't work, and what I actually learned. I'm not
posting these because I have it all figured out I'm posting them because I think
documenting your process is how you actually grow, and because I want anyone looking at this
to see how I think, not just that I got the flag.
